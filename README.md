# VERA Interiors Website

Your new website for VERA Interiors is ready!

## How to View the Website
1. Open the folder `VERA_Interiors` on your Desktop.
2. Double-click the `index.html` file.
3. The website will open in your default web browser.

## Features Included
- **Home Page**: Modern hero section, services preview, testimonials.
- **About Us**: Team introduction and company story.
- **Services**: Detailed pages for Apartment, House, and Restaurant interiors.
- **Portfolio**: Filterable gallery of high-quality images.
- **Contact**: Functional contact form layout and location details.
- **Mobile Responsive**: Works on phones and tablets.
- **Animations**: Smooth scroll and fade-in effects.

## Technologies Used
- HTML5
- Tailwind CSS (via CDN)
- JavaScript (Vanilla)
- Unsplash Images (High Quality)

Enjoy your new site!
